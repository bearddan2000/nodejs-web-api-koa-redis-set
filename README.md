# nodejs-web-api-koa-redis-set

## Description
A basic nodejs api with koa
framework that uses redis
to cache a set requests.

## Tech stack
- nodejs
  - koa
  - json
  - promise
- redis

## Docker stack
- node:latest
- redis:latest

## To run
`sudo ./install.sh -u`
- curl -X localhost

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
